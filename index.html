Introduction
This is a Automation Design Solidworks Tool (Solidworks 2018) use for Mechanical Engineer for extremely fast building 3D standard mold die set.
Quick start

Deployment
1. Clone the project by git
2. Navigate to DieSetTool\bin\Release
3. Run DieSetTool.exe
Benefit
- Saving time
- In traditional way, takes 16 hours to design 1 die set
- Using tool, export total die set takes 5 mins
- Keep sanity, no boring and discouragement anymore
- Free coffee time when waiting
Tool stack:
C# - Language
Microsoft Visual Studio 2015 - Framework
Solidwo rks API - Provide by Dassault Systemes to use solidworks functions
